 Task 3: LCD Display using Arduino (I2C)

 Aim
To interface a 16×2 I2C LCD with Arduino Uno and display text.

 Components Used
- Arduino Uno  
- 16×2 I2C LCD  
- Jumper wires  

 Connections
| LCD Pin | Arduino Pin |
|--------|-------------|
| GND    | GND         |
| VCC    | 5V          |
| SDA    | A4          |
| SCL    | A5          |

 Logic
- Arduino communicates with the LCD using I2C protocol.
- The LCD is initialized with its I2C address.
- Text is displayed using Arduino code.

 Output
The LCD displays text successfully in TinkerCAD simulation.
