 ESM Arduino Tasks

This repository contains three Arduino tasks implemented using TinkerCAD as part of the ESM assignment.

Tasks

1. LED Brightness Control using Potentiometer
   Folder: `led-brightness`  
   - Controls the brightness of an LED using a potentiometer and PWM.
   - https://www.tinkercad.com/things/5hmwfWtjx11-led?sharecode=tDQo4YikJpWDf4f8DlJGorTW64Qs4Xi8S46cIm84O9A

2. RGB LED Control using Potentiometers
   Folder: `rgb-led`  
   - Controls the Red, Green, and Blue intensities of an RGB LED using potentiometers.
   - https://www.tinkercad.com/things/8IqZuFnlJti-merinrgb?sharecode=jMmhKi4Oq6qV6FnY5Sw8kDMu6fBbHSY7JA7d99uilEk

3. LCD 16×2 Display using Arduino (I2C)
   Folder: `lcd-display`  
   - Displays text on a 16×2 LCD using I2C communication.
   - https://www.tinkercad.com/things/74Q0yhh5XtQ-super-curcan-crift?sharecode=fSdzaTa_MBbzQjVeDZuUHKDS8cNfTQ1g00Ys_YpAimo

Each task folder contains:
- A `README.md` explaining the logic
- Arduino source code
- Circuit screenshot from TinkerCAD
